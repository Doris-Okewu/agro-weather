AGROWEATHER by Doris Okewu

HOW TO RUN:
1. Open the "AgroWeather_DIST" folder
2. Double-click "index.html"
3. Type any city (e.g. Lagos) → enjoy!

FEATURES:
- Current weather
- 5-day forecast
- Farming tips for crops
- Add your own crop
- Dark mode
- Works on phone & laptop

NOTE:
- Uses mock data for demo (no internet needed)
- Real weather API ready for production
